# Architecture Decision Records (ADR)

## Purpose
ADRs capture important technical decisions and their rationale.

## ADR Lifecycle
Proposed -> Review -> Accepted -> Superseded -> Deprecated

## ADR Template

### ADR-XXXX Title

Status: Proposed

Context:
- Problem statement
- Constraints
- Requirements

Options Considered:
1. Option A
2. Option B
3. Option C

Decision:
Selected option and justification.

Consequences:
Positive:
- ...

Negative:
- ...

Implementation Notes:
- ...
