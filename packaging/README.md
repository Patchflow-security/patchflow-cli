# Packaging

PatchFlow release packages are generated by GoReleaser from `.goreleaser.yml`.

Published channels:

- GitHub release archives and checksums
- Homebrew formula in `Patchflow-security/homebrew-tap`
- Scoop manifest in `Patchflow-security/scoop-bucket`
- GHCR image `ghcr.io/patchflow-security/cli`

Do not maintain manual Homebrew or Scoop manifests in this repository; stale
hashes and hand-written formulas can diverge from signed release artifacts.
